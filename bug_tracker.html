This page provides information for users who would like to report a bug or submit a feature request.

Bug Reports:

Bug reports should be as detailed a possible to help developers try to reproduce and fix them. In order to help gather information that could help the bug fix process, please use this template when submitting a bug report.

Bug reports should be submitted through the issue tracker on the github website.



Feature Requests:

As with bug reports, feature requests should be as detailed as possible. They should also be submitted through the issue tracker on the github website.
