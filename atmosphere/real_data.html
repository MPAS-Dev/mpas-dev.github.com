Sample real-data input files

The MPAS-Atmosphere Users' Guide describes the process of setting up MPAS-Atmosphere for real-data simulations, which feature a realistic representation of the earth's surface and are typically initialized with gridded atmospheric analyses. Currently, MPAS-Atmosphere relies on data files and programs from the WRF Pre-processing System to describe the surface characteristics and to process atmospheric analyses in GRIB format.

For the purposes of code testing, the following downloads are provided so that users can avoid the work of downloading and installing the WPS and its associated static datasets, and proceed directly to running an example real-data case on the 40962 mesh (approximately 120-km grid spacing) initialized from the CFSR analysis valid 2010-10-23 at 00 UTC.

  • A "static" file, x1.40962.static.nc,
  • an intermediate file produced by ungrib.exe from the CFSR analysis, CFSR:2010-10-23_00, and
  • namelist and stream configuration files for running init_atmosphere_model and atmosphere_model, namelist_files.tar.gz.
