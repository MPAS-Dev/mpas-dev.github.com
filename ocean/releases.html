MPAS-Ocean Releases


MPAS-Ocean releases are listed below. Please ensure you are using a combination of user's guide, input file, source code, and api reference for the same version number as information can change between versions.

Before you begin, please fill out the MPAS Download Registration. For information about citing MPAS-Ocean please see MPAS-Ocean Information.

For the latest release, check the MPAS github tag page. Incremental releases may not be listed in the following table. MPAS-Ocean does not participate in all releases.

NOTE:A specific version below applies to all subsequent versions in the series (e.g. 1.x) unless another version in that series is listed below. For example, version 1.0 applies to all 1.x version, but if version 1.5 is listed, it applies to all versions between 1.5 and 2.0.
